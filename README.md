# World Cup 2026 — Circular Bracket

A standalone, dependency-free page showing a circular FIFA World Cup 2026 knockout bracket and a FIFA-ranking "Next Matches" chart. Pure HTML/CSS/JS, no build step, no backend.

Click the trophy (or load the page) to download the latest results from [openfootball/worldcup.json](https://github.com/openfootball/worldcup.json).

## Running locally

Just open `index.html` in a browser, or serve the folder with any static file server, e.g.:

```bash
npx serve .
```
